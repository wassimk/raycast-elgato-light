# Elgato Light Changelog

## [Initial Version] - {PR_MERGE_DATE}

- Auto-discovery of Elgato lights via Bonjour/mDNS
- Manual IP address configuration for networks where auto-discovery is unreliable
- Multi-light support for controlling all lights simultaneously
- Day Light and Night Light presets with configurable brightness and temperature
- Set exact color temperature (2900K-7000K)
- Adjustable brightness increment amount
- Light discovery caching with manual cache clearing
